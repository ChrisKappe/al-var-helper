VS Code extension for variable handling in AL.

### Features
- Records
- Codeunits
- Reports
- Queries
- Pages

### Usage
To create a new variable type t + <Object Type> (e.g. trecord).
To select a name for the variable tab to the name and select a name with Ctrl + Space
